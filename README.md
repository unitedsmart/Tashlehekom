# تطبيق تشليحكم - Tashlehekomv2

تطبيق Flutter لبيع وشراء السيارات وقطع الغيار المستعملة.

## المواصفات

### الميزات الرئيسية

- **عرض السيارات بدون تسجيل دخول**: يمكن للمستخدمين تصفح السيارات مباشرة
- **أنواع المستخدمين المختلفة**:
  - فرد (مشتري/بائع)
  - عامل في التشليح
  - مالك تشليح
- **نظام المصادقة**: تسجيل دخول بـ OTP عبر رقم الجوال
- **إدارة السيارات**: رفع وعرض السيارات مع الصور والتفاصيل
- **نظام التقييم**: تقييم البائعين بالنجوم
- **لوحة الإدارة**: إدارة المستخدمين والموافقات
- **التواصل**: اتصال وواتساب (للمستخدمين المسجلين فقط)
- **الخرائط**: عرض موقع السيارة (للمستخدمين المسجلين فقط)

### التقنيات المستخدمة

- **Flutter**: إطار العمل الأساسي
- **SQLite**: قاعدة البيانات المحلية
- **Provider**: إدارة الحالة
- **Google Maps**: عرض المواقع
- **Image Picker**: اختيار الصور
- **URL Launcher**: فتح الروابط والاتصال

## التثبيت والتشغيل

### المتطلبات

- Flutter SDK (3.0.0 أو أحدث)
- Dart SDK
- Android Studio أو VS Code
- جهاز Android أو iOS للاختبار

### خطوات التثبيت

1. **استنساخ المشروع**:
```bash
git clone <repository-url>
cd tashlehekomv2
```

2. **تثبيت التبعيات**:
```bash
flutter pub get
```

3. **إضافة الأصول**:
   - ضع ملفات الخطوط في `assets/fonts/`
   - ضع الصور في `assets/images/`

4. **تشغيل التطبيق**:
```bash
flutter run
```

## الاستخدام

### للمستخدمين العاديين

1. **تصفح السيارات**: يمكن تصفح السيارات بدون تسجيل دخول
2. **البحث والفلترة**: استخدم شريط البحث والفلاتر للعثور على السيارة المناسبة
3. **تسجيل الدخول**: للتواصل مع البائعين أو إضافة سيارات
4. **إضافة سيارة**: اضغط على زر "أضف سيارة أو قطعة"

### للإدارة

1. **الوصول للوحة الإدارة**: يتطلب صلاحيات خاصة
2. **الموافقة على الحسابات**: مراجعة وتفعيل حسابات مالكي التشاليح
3. **إدارة العمال**: ربط العمال بالتشاليح
4. **عرض الإحصائيات**: متابعة أداء التطبيق
