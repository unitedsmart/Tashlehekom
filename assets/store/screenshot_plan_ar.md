# خطة لقطات الشاشة (الهاتف 1080×1920)

التقاط مباشر من التطبيق (واجهة عربية RTL). افتح كل شاشة بالمحتوى الظاهر كما في النقاط:

1) الشاشة الرئيسية (Home)
- الهيدر الأخضر (تشليحكم + الشعار)
- البانر العلوي
- بطاقات "إحصائيات سريعة"
- "إجراءات سريعة" (شراء قطع غيار / بيع سيارة)

2) البحث المتقدم (Search)
- SearchFilterBar ظاهر بالكامل
- نتائج قائمة السيارات (CarCard) بعد تحميل بيانات تجريبية

3) تفاصيل السيارة (CarDetails)
- معرض الصور في الأعلى
- بطاقة "تفاصيل السيارة"
- بطاقة "معلومات البائع"
- أزرار (اتصال) و(واتساب)

4) قطع الغيار (Parts)
- حقل بحث + فلتر مدينة
- بطاقات قطع تتضمن السعر بالريال والمدينة والصورة

5) الدردشة (EnhancedChat)
- محادثة مفتوحة مع اسم المستخدم وعنوان السيارة
- فقاعات رسائل واضحة (يمين/يسار)

6) الإشعارات أو المفضلة
- شاشة Notifications أو قائمة مفضلة إن متاحة

نصوص Overlay المقترحة (أقصى 2 سطر لكل صورة):
- الرئيسية والبحث الذكي
- فلاتر دقيقة حسب الماركة والموديل
- تفاصيل السيارة وصور متعددة
- قطع غيار بأسعار واضحة
- دردشة وإشعارات فورية
- إدارة الإعلانات والمفضلة

نصائح:
- أوقف شريط الحالة إذا أمكن أو استخدم Emulator نظيف
- فعّل الوضع العادي (غير ليلي) لثبات الألوان
- احفظ كل صورة ≤ 8MB وبدقة 1080×1920 PNG/JPG
