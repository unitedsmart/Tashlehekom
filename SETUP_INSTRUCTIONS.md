# تعليمات إعداد وتشغيل تطبيق تشليحكم

## المتطلبات الأساسية

### 1. تثبيت Flutter
```bash
# تحميل Flutter SDK من الموقع الرسمي
# https://flutter.dev/docs/get-started/install

# إضافة Flutter إلى PATH
export PATH="$PATH:`pwd`/flutter/bin"

# التحقق من التثبيت
flutter doctor
```

### 2. إعداد Android Studio
- تحميل وتثبيت Android Studio
- تثبيت Android SDK
- إنشاء Android Virtual Device (AVD)

## خطوات التشغيل

### 1. تحضير المشروع
```bash
# الانتقال إلى مجلد المشروع
cd tashlehekomv2

# تثبيت التبعيات
flutter pub get

# تنظيف المشروع (اختياري)
flutter clean
```

### 2. إضافة الأصول المطلوبة

#### الخطوط
- تحميل خط Cairo من Google Fonts
- وضع الملفات في `assets/fonts/`:
  - `Cairo-Regular.ttf`
  - `Cairo-Bold.ttf`

#### الصور
- إضافة الصور التالية في `assets/images/`:
  - `logo.png` (شعار التطبيق)
  - `header_banner.jpg` (صورة الرأس)

### 3. إعداد Google Maps (اختياري)
```bash
# الحصول على Google Maps API Key من Google Cloud Console
# استبدال YOUR_GOOGLE_MAPS_API_KEY_HERE في AndroidManifest.xml
```

### 4. تشغيل التطبيق
```bash
# تشغيل على جهاز متصل أو محاكي
flutter run

# تشغيل في وضع التطوير مع Hot Reload
flutter run --debug

# بناء APK للإنتاج
flutter build apk --release
```

## اختبار التطبيق

### بيانات الاختبار
- **رمز OTP للاختبار**: 1234
- **أرقام جوال للاختبار**: أي رقم يبدأ بـ 05

### سيناريوهات الاختبار

#### 1. تصفح السيارات (بدون تسجيل دخول)
- فتح التطبيق
- تصفح السيارات المعروضة
- استخدام البحث والفلاتر
- عرض تفاصيل السيارة

#### 2. تسجيل مستخدم جديد
- الضغط على "تسجيل الدخول"
- اختيار "إنشاء حساب جديد"
- اختيار نوع المستخدم
- إدخال البيانات المطلوبة
- التحقق من OTP (استخدم 1234)

#### 3. إضافة سيارة
- تسجيل الدخول كمستخدم مفعل
- الضغط على "أضف سيارة أو قطعة"
- ملء بيانات السيارة
- إضافة الصور
- حفظ السيارة

#### 4. لوحة الإدارة
- تسجيل الدخول كمدير
- الوصول للوحة الإدارة
- مراجعة الحسابات قيد الموافقة
- إدارة العمال
- عرض الإحصائيات
