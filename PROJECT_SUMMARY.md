## الهيكل التقني

### قاعدة البيانات
- **SQLite** للتخزين المحلي
- جداول: المستخدمين، السيارات، التقييمات، الماركات، المدن
- علاقات مترابطة مع Foreign Keys

### إدارة الحالة
- **Provider** لإدارة الحالة
- مقدمي خدمات منفصلين للمصادقة والسيارات والمستخدمين

### واجهة المستخدم
- **Material Design** مع ثيم مخصص
- دعم اللغة العربية
- تصميم متجاوب
- خط Cairo العربي

### التبعيات الرئيسية
- `sqflite`: قاعدة البيانات
- `provider`: إدارة الحالة
- `image_picker`: اختيار الصور
- `google_maps_flutter`: الخرائط
- `url_launcher`: التواصل
- `flutter_rating_bar`: التقييم
- `carousel_slider`: عرض الصور
